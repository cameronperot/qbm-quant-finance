qbm.plotting.metrics.plot_autocorrelation(ax, lags, acf, title, **kwargs)

Plots the autocorrelation (acf) as a function of the lag on ax.

Parameters

  • ax – Matplotlib axis.

  • lags – Lags over which the acf is computed.

  • acf – Autocorrelation function.

  • title – Title of the plot.

Returns

Matplotlib axis.

qbm.plotting.metrics.plot_correlation_coefficients(data, sample)

Plots the data correlation coefficients against those of the sample.

Parameters

  • data – Correlation coefficients of the data.

  • sample – Correlation coefficients of the sample, this is a dict of dataframes, containing keys [“means”, “stds”] (e.g. output of compute_stats_over_dfs).

Returns

Matplotlib figure and axes.

qbm.plotting.metrics.plot_qq(ax, data, sample, title, params, **kwargs)

Plots a QQ plot of the data against the sample on the provided ax.

Parameters

  • ax – Matplotlib axis.

  • data – Array of data points.

  • sample – Array of sample.

  • title – Title of the plot.

  • params – Additional parameter dictionary for ax configuration, required keys are [“xlims”, “ylims”, “xticks”, “yticks”].

Returns

Matplotlib axis.

qbm.plotting.metrics.plot_qq_grid(data, sample, params)

Plots a 2x2 grid of QQ plots.

Parameters

  • data – A dataframe of shape (n_sample, 4).

  • sample – A dataframe with matching column names to the data and the same shape.

  • params – Additional parameter dictionary for ax configuration, required keys are [“xlims”, “ylims”, “xticks”, “yticks”].

Returns

Matplotlib figure and axes.

qbm.plotting.metrics.plot_tail_concentrations_grid(dfs, combinations, colors, interval_size=0.001)

Plots the upper and lower tail concentration functions for the provided dataframes.

Parameters

  • dfs – Dictionary where keys are the plot labels, and the values are the corresponding dataframes to be plotted.

  • combinations – List of tuples of column names to be plotted, e.g. [(“EURUSD”, “GBPUSD”), (“EURUSD”, “USDJPY”), …]

  • colors – Dictionary where the keys are the plot labels, and the values are colors.

  • interval_size – Spacing between the values on the independent axis.

Returns

Matplotlib figure and axes.

qbm.plotting.metrics.plot_volatility(ax, dates, volatility, title, params)

Plots the historical volatility against the date, as well as a horizontal line indicating the median value over the entire historical period.

Parameters

  • ax – Matplotlib axis.

  • dates – Array of dates to plot against.

  • volatility – Array of volatility values to plot against the dates.

  • title – Title of the plot.

  • params – Additional parameter dictionary for ax configuration, required keys are [“xlims”, “ylims”, “xticks”, “yscale”, “label”].

Returns

Matplotlib axis.

qbm.plotting.metrics.plot_volatility_comparison(data, sample, params)

Plots the data annualized volatility against those of the sample on an error bar plot.

Parameters

  • data – Annualized volatility of the data.

  • sample – Annualized volatility of the sample, this is a dict of series, containing keys [“means”, “stds”] (e.g. output of compute_stats_over_dfs).

  • params – Additional parameter dictionary for ax configuration, required keys are [“xlims”, “ylims”, “yticks”].

Returns

Matplotlib figure and axes.

qbm.plotting.metrics.plot_volatility_grid(volatility, params)

Parameters

  • volatility – Dataframe of historical volatilities.

  • params – Additional parameter dictionary for ax configuration, required keys are [“xlims”, “ylims”, “xticks”, “yscale”, “label”].

Returns

Matplotlib figure and axes.
