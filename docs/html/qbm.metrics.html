qbm.metrics.metrics.compute_correlation_coefficients(df, combinations, sample_number=None)

Computes the Pearson r, Spearman r, and Kendall τ coefficients for all combinations of columns in the provided dataframe.

Parameters

  • df – Dataframe.

  • combinations – List of tuples of column name pairs to compute the correlations of.

  • sample_number – If sample number is provided, then an additional column will be appended to the dataframe corresponding to the sample number.

Returns

Dataframe where rows are the combinations and columns are the correlation coefficients.

qbm.metrics.metrics.compute_rolling_volatility(df, window, drop_insufficient_data_rows=True)

Compute the rolling volatility of the input dataframe over the provided window.

Parameters

  • df – Dataframe of which to compute the volatility from.

  • window – Window over which to compute the volatility, e.g. timdelta(days=90).

  • drop_insufficient_data_rows – If True will drop rows that do not have enough data to fill the window.

Returns

Dataframe of the rolling volatilities.
