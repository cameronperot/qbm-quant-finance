qbm.sampling.bernoulli_rbm module

qbm.sampling.bernoulli_rbm.generate_rbm_sample(model, v, n_steps)

Generate a sample after a number of passes from the provided RBM model from the initial state v.

Parameters

  • model – Scikitlearn RBM model.

  • v – Initial state to start the visible layer in.

  • n_steps – Number of passes through the network to perform, i.e., the number of Gibbs sampling steps.

Returns

The visible layer after n_steps Gibbs sampling steps.

qbm.sampling.bernoulli_rbm.generate_rbm_samples_df(model, v, n_samples, n_steps, model_params)

Generate a dataframe of samples.

Parameters

  • model – Scikitlearn RBM model.

  • v – Initial state to start the visible layer in.

  • n_samples – Number of samples to generate.

  • n_steps – Number of passes through the network to perform, i.e., the number of Gibbs sampling steps.

  • model_params – Dict with required keys [“n_bits”, “columns”, “binarization_params”, “split_indices”]

Returns

Dataframe of samples, final visible state.
